# Expensy_1.0 💰

A Python-based Expense Tracker application that helps users manage daily expenses, generate reports, and visualize spending trends.

## Features

* User Registration & Login
* Add Expenses
* View Expenses
* Update Expenses
* Delete Expenses
* Search Expenses by Category
* View Expenses by Date
* Today's Expense Summary
* Monthly Expense Reports
* Category-wise Expense Reports
* Export Monthly Reports to CSV
* Highest & Lowest Expense Analysis
* Category-wise Pie Chart
* Monthly Expense Trend Chart

## Technologies Used

* Python
* CSV File Handling
* Datetime Module
* Matplotlib

## Project Structure

```text
Expensy_1.0/
│
├── expensy.py
├── users.csv
└── user_expense_files.csv
```

## How to Run

1. Install Python 3.x
2. Install matplotlib

```bash
pip install matplotlib
```

3. Run the project

```bash
python expensy.py
```
